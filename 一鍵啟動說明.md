# 田徑競賽系統：開發機一鍵啟動

## Windows 開發機

直接雙擊：

```text
start-dev.cmd
```

或用 PowerShell 執行：

```powershell
.\start-dev.ps1
```

預設網址：

- 前端平台：`http://localhost:4200`
- API 健康檢查：`http://localhost:3000/api/health`

如果第一次執行找不到 `server/.env`，腳本會自動建立一份本機測試用設定。

## Linux / 遠端開發機

```bash
cd /home/derek/track-system
./start-dev.sh
```

遠端腳本預設使用正式服務的 API Port `3100`。如果 `http://127.0.0.1:3100/api/health` 已正常，腳本會直接沿用既有後端，只啟動前端測試站；如果 API 沒有啟動，才會自動補裝後端開發套件並啟動後端。

預設網址：

- 前端平台：`http://172.238.19.47:4200`
- API 健康檢查：`http://172.238.19.47:3100/api/health`
- 後端紀錄：`.dev-logs/server.log`
- 前端紀錄：`.dev-logs/client.log`

按 `Ctrl+C` 可關閉本腳本啟動的前端與後端。

## 常用參數

Windows PowerShell：

```powershell
.\start-dev.ps1 -ApiPort 3100 -WebPort 4201
.\start-dev.ps1 -Install
.\start-dev.ps1 -NoBrowser
```

Linux：

```bash
API_PORT=3100 WEB_PORT=4201 ./start-dev.sh
INSTALL=1 ./start-dev.sh
```

## 注意事項

- 請先安裝 Node.js 20 以上版本。
- 請確認 MongoDB 已啟動；預設資料庫連線為 `mongodb://127.0.0.1:27017/track-system`。
- 如果 Port 已被占用，請改用上方參數切換 Port。
